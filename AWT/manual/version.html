    • Changed the order in the match-list, the most recent match is now displayed at the top

  • 97-10-10, v1.06
    • Added possibility to use dates in the matchfile

  • 97-08-28
    • The German version is released
  • 97-07-29
    • The Finnish version is released
  • 97-05-28, v1.05
    • MarkX, new parameter for marking a team in the table
    • Better handling of multiple leagues
    • Some minor bugfixes
  • 97-03-15
    • The French version is released
  • 97-02-12, v1.04
    • Can now show meetings between two teams from previous seasons
    • New JavaScript-function: loadLeaguefile()
    • LeagueName and Season, new parameters for giving the name and season of the league
    • HideBorder, parameter for "hiding" the border that is drawn around the applet
    • Fixed bug that caused incorrect screen-updates in Netscape for UNIX
  • 97-01-19, v1.03
    • Some functions can now be controlled via JavaScript
    • Instead of using <PARAM> you can set parameters in a parameterfile
    • Added tip about archivefiles
    • Some additional visual changes
    • Fixed bug that sometimes caused the program to load incorrectly
    • Fixed bug that appeared when teams were hidden with the HideX-parameter
  • 96-12-08, v1.02
    • TeamStats is now available in two new languages: Spanish and Portuguese
    • PointsX, new parameter for giving teams bonus- or penaltypoints
    • HideX, new parameter to "hide" teams that are degraded or promoted during the season
    • Smaller font in the main buttons
    • Better adjustment to sports with many goals (points), such as bowling and basket
  • 96-11-18, v1.01
    • Added a few error messages
    • Results that has occurred only one time does not show in "Most frequent result"
  • 96-10-26, v1.0

  • 96-10-07, Version 1 Beta 4
    • Made the program more general to suit more sports
