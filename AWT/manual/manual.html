TeamStats

Brief manual

The program has four main functions:

  • Table
  • Matches
  • Statistics
  • Compare teams

Table

You decide how the table will be displayed by changing the table settings. In the box Matches you can decide if the table should be a home, away or total table. In the Rounds box you decide if the table should be over all, the first or the last rounds. If you choose first or last you can also enter the number of rounds. A table over the five last rounds is a good way to compare the form of the teams. You can also combine the settings for Matches and Rounds and for example display a table over the five last home matches. To show the current table just select All in both Matches and Rounds.

Matches

The list with matches is displayed backwards, with the most recent match at the top. You can either see all the played matches or select a team and see only the matches of that team. To see all the matches you select All in the Team box. If you select a team you can then select if you want to see all, home or away matches in the Matches box. In the Results box you select if you want to see won, drawn or lost matches. When you have selected a team the matches will be displayed in a different format. After the opponents name you see an (H) or an (A). (H) means that the selected team played home and (A) that they played away. The result that follows is always displayed with the selected teams goals first regardless of if they played home or away.

Statistics

Here you can see some miscellaneous stats from the league. Hopefully it's quite self-explanatory.

Compare teams

In this section you can select two teams and compare their stats prior to a match between them. You select the teams in the boxes Hometeam and Awayteam. In the field above the team boxes you select if you want to see Statistics, Home/away stats or Previous matches.

In Statistics you first see some information about the table position, played matches and goalstats etc. Then the five last matches are displayed. The most recent match is showed at the top of the list. In Current streaks you can see if the teams have any streaks running. For example a team can have played five matches in a row without losing or without scoring a goal. At the bottom the teams most frequent results are displayed.

In Home/away stats the same information as in Statistics are showed, limited to home- and awayresults. Facts about the home team are only based on their home matches and the other way around for the away team.

In Previous matches the results from previous matches between the two teams are shown.
